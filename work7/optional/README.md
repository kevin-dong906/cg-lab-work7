# SMPL 关节旋转动画与 LBS 权重平滑带动效果实验报告
## 一、实验目标
1. 固定 SMPL 形状参数 beta，保持人体体型不变，仅通过单关节连续旋转生成姿态动画。
2. 实现左肘关节（关节 18）从 0 逐步弯曲到最大角度的动画，生成连续帧图像与 GIF 动画。
3. 直观观察 LBS 蒙皮权重如何平滑带动皮肤网格随骨骼运动，验证线性混合蒙皮的平滑过渡特性。
4. 掌握基于 SMPL 模型的关键帧动画生成、连续姿态求解、3D 网格序列可视化方法。

## 三、实验原理
### 1. 固定形状参数
形状参数在整个动画过程中保持不变，人体体型固定，不发生高矮胖瘦变化。

### 2. 单关节连续旋转
选择左肘关节（索引 18），在 0 到最大角度之间进行线性插值。

角度计算公式：
```
angle(i) = max_angle * i / (N - 1)
```
N 为总帧数，本实验 N = 30。

### 3. SMPL 模型前向计算
每一帧使用不同姿态参数，调用 SMPL 模型得到当前姿态的顶点与关节坐标。

### 4. LBS 权重平滑带动原理
顶点最终坐标由多个关节的变换加权求和得到：
```
v' = w1*G1*v + w2*G2*v + ... + wK*GK*v
```
权重越高的区域，跟随关节运动越明显；权重越低则几乎不动，从而实现平滑自然的皮肤形变。

## 四、代码结构与关键模块分析
### 1. 整体架构
代码分为模型兼容层、绘图渲染层、动画生成层、主逻辑层，结构清晰可直接运行。

### 2. 关键代码解析
#### （1）固定形状参数
```python
betas = torch.zeros((1,10), device=device, dtype=dtype)
betas[0,0] = 2.0
betas[0,1] = -1.2
betas[0,2] = 0.8
```
功能：保持体型不变，确保动画只体现姿态变化。

#### （2）单关节连续旋转
```python
angle = max_angle * (i / (total_frames-1))
pos = (joint_idx -1)*3
body_pose[0, pos:pos+3] = torch.tensor([0.0, -angle, 0.0], dtype=dtype)
```
功能：对左肘关节沿 Y 轴旋转，角度从 0 线性增加。

#### （3）逐帧渲染与 GIF 合成
```python
save_frame(p, v, faces, j)
images.append(Image.open(p))
images[0].save(gif_path, save_all=True, append_images=images[1:], duration=60, loop=0)
```
功能：保存每一帧图像，最后合并为循环播放的 GIF。

## 五、实验步骤
1. 加载 SMPL 中性模型，固定形状参数。
2. 设置动画总帧数 30 帧，左肘关节从 0 旋转到 1.2 弧度。
3. 逐帧计算 SMPL 顶点与关节位置。
4. 逐帧渲染 3D 人体网格并保存图片。
5. 将所有帧合成 GIF 动画，观察权重带动效果。

## 六、实验结果
### 1. 文件输出
- 30 帧连续 PNG 图像：anim_frames/frame_00.png ~ frame_29.png
- 循环动画：animation.gif

### 2. 视觉现象
1. 体型全程不变，人体轮廓、比例无变化。
2. 肘关节平滑弯曲，从伸直到弯曲无跳变。
3. 权重区域平滑带动：
   - 小臂、手肘附近运动最明显
   - 上臂运动幅度次之
   - 躯干、肩膀、腿部几乎不动
   - 过渡区域自然柔软，无断裂、无穿模、无生硬棱角

### 3. 动画观察结论
- LBS 权重分布决定皮肤跟随骨骼运动的强度。
- 权重平滑分布 → 形变平滑过渡。
- 单关节运动可带动一片连续区域，符合真实人体运动规律。

## 七、实验总结
1. 成功实现固定体型、单关节连续旋转的 SMPL 姿态动画。
2. 生成 30 帧平滑动画与 GIF，直观展示 LBS 蒙皮的权重带动效果。
3. 验证了线性混合蒙皮平滑、自然、无撕裂的形变特性。
4. 理解关节旋转 → 权重加权 → 网格形变的完整流程。
5. 掌握 SMPL 动画生成、逐帧渲染、GIF 导出的工程方法。

本实验清晰展示了 LBS 蒙皮的核心优势：基于权重的平滑带动，是数字人、角色动画的基础关键技术。
